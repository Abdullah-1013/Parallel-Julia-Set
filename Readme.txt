## Features

- Real-time Julia Set generation
- Adjustable real and imaginary values
- Fast backend using OpenMP
- Save generated images
- Generate fractals resembling tree or mountain forms

---

## Installation
- install python
- Install Library:
  pip install pillow numpy

## How to run Porject
- To compile Backend:
   gcc -fopenmp -o julia_openmp.exe julia_openmp.c
- To use GUI:
  python julia_openmp.py
